Supplementary Material for `Biclustering Models for Structured Microarray Data'

The supplementary material for `Biclustering Models for Structured Microarray Data' (Turner et al, IEEE/ACM Trans. Comp. Biol. Bioinf., 2005) provides the R code used to produce the results presented in the paper, based on the algorithm presented in `Improved Biclustering of Microarray Data Demonstrated through Systematic Performance Tests' (Turner et al, Comput. Statist. Data Anal., 2005). A copy of this material is provided here.

It should be noted however that this code is not fully documented and is not maintained. I would direct interested readers instead to the biclust package for R, which provides a number of biclustering algorithms, including a plaid function based on our algorithm.
